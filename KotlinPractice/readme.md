<h1>Jetpack Compose UI</h1>

<h2>Layouts</h2>
<p>There are three commonly used layouts which are Row Layout, Column Layout and Box Layout.</p>
<ol>
  <li>Row Layout
  <pre>
      <code>
        Row (
          horizontalArrangement = Arrangement.Center,
          modifier = Modifier.fillMaxSize()
        ){
          Text(text = "Row1.1")
          Text(text = "Row1.2")
        }
      </code>
    </pre>
  </li>
</ol>
